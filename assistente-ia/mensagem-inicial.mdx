Os primeiros segundos de uma chamada são cruciais - eles determinam se o cliente permanece na linha. Você tem duas opções para a primeira saudação do seu assistente: mensagem inicial baseada em texto ou áudio inicial personalizado.

## Mensagem Inicial

Esta é a primeira coisa que seu assistente de IA diz ao iniciar uma chamada. A mensagem é lida exatamente como escrita, então:

### Melhores Práticas

1. **Mantenha Curta**

   - Mire em 5-10 segundos
   - Vá direto ao ponto rapidamente
   - Evite apresentações longas da empresa

2. **Escreva Exatamente Como Necessário**

   - Inclua diacríticos adequados (é, ñ, ü, etc.)
   - Use pontuação para pausas adequadas
   - Escreva números como devem ser falados

3. **Formatos de Exemplo**

   ```
   Bom:
   "Olá! Aqui é a Sarah da Empresa ABC. Como posso ajudá-lo hoje?"

   Melhor com pausas:
   "Olá! Aqui é a Sarah da Empresa ABC... Como posso ajudá-lo hoje?"

   Com diacríticos:
   "¡Hola! Soy María de ABC Company. ¿Cómo puedo ayudarte?"
   ```

## Áudio Inicial

Para a melhor primeira impressão, você pode usar um arquivo de áudio pré-gravado:

### Benefícios

- Qualidade profissional
- Pronúncia perfeita
- Calor humano
- Entrega consistente
- Maior retenção de clientes

### Processo de Configuração

1. Grave sua saudação com um ator de voz profissional
2. Faça upload do arquivo de áudio nas configurações do assistente
3. Clone a mesma voz para o resto da conversa
4. Ative a reprodução de áudio inicial

### Melhores Práticas

1. **Qualidade de Gravação**

   - Use equipamentos profissionais
   - Grave em ambiente silencioso
   - Mantenha volume consistente
   - Salve em formato de alta qualidade

2. **Correspondência de Voz**

   - Use o mesmo ator de voz para clonagem
   - Mantenha tom e estilo consistentes
   - Corresponda níveis de energia

3. **Diretrizes de Conteúdo**
   - Mantenha abaixo de 10 segundos
   - Inclua nome da empresa
   - Declare o propósito claramente
   - Soe acolhedor

### Exemplo de Estrutura de Script

```
[Saudação] + [Nome da Empresa] + [Propósito/Pergunta]

"Olá! Aqui é a Empresa ABC ligando sobre sua consulta recente. Como você está hoje?"
```

## Combinando Ambos os Métodos

Você pode configurar ambos:

- Áudio inicial como saudação primária
- Mensagem inicial como backup
- Sistema usará áudio quando disponível

## Testes

Antes de entrar no ar:

1. Ligue para seu assistente
2. Escute por:
   - Pronúncia clara
   - Pausas naturais
   - Volume adequado
   - Transição suave para conversa da IA

## Considerações de Idioma

- Cada idioma precisa de sua própria mensagem/áudio inicial
- Use falantes nativos para gravações
- Considere sotaques regionais
- Teste com público-alvo
