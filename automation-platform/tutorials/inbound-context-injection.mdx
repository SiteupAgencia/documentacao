# Injeção de Contexto para Chamadas Recebidas

Este tutorial demonstra como aprimorar as conversas do seu assistente de IA injetando contexto em tempo real antes da chamada se conectar.

## O Que Você Aprenderá

- Configurar o trigger de Chamada Recebida
- Buscar dados de produtos mais recentes do WooCommerce
- Estruturar contexto para a IA

## Pré-requisitos

- Conta Clinicalls com números de entrada
- Integração com WooCommerce
- Assistente de IA configurado para injeção de contexto
