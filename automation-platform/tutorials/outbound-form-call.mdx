# Chamada de Saída no Envio de Formulário

Este tutorial mostra como configurar um fluxo de automação que dispara uma chamada de IA quando um cliente envia um formulário em seu site.

## O Que Você Aprenderá

- Configurar um trigger de envio de formulário
- Configurar a ação Clinicalls
- Mapear dados do formulário para variáveis da chamada
- Testar e implantar seu fluxo
