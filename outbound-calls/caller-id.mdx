Quando você lança uma campanha de saída, pode escolher qual número aparece nos telefones dos destinatários.

## 1. Número Dedicado

- Se você alugou um número local ou gratuito, pode selecioná-lo como "Número de telefone" nas configurações do assistente.
- Bom para consistência da marca ou se você também aceita chamadas de entrada na mesma linha.
- **Nenhuma verificação OTP necessária** para números alugados através do Autocalls.ai.

## 2. Número Existente Verificado (Apenas ID do Chamador)

- Se você quer que sua linha telefônica pessoal ou da empresa apareça, verifique-a com um **OTP** ou chamada curta.
- Isso não lidará com chamadas de entrada para a IA, mas o lead vê esse número reconhecido no ID do chamador.
- **Nota**: A verificação OTP é necessária APENAS quando usar seu próprio número existente como ID do Chamador.

## 3. Integração SIP

- Se você usa uma plataforma PBX ou VOIP, pode passar um ID do chamador personalizado do seu sistema existente.
- Certifique-se de que seu tronco ou provedor esteja configurado para permitir um ID do Chamador de saída específico.

## 4. Regras Específicas do País

- Algumas regiões impedem o uso de números que você não possui como ID do Chamador.
- Sempre certifique-se de ter o direito de apresentar esse número para evitar problemas de spam ou legais.

---

**Dica:** Teste ligando para seu próprio telefone ou um colega amigável para confirmar que o ID do chamador correto é exibido antes de lançar uma campanha grande.
