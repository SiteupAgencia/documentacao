Depois de adquirir um número (ou configurar SIP ou ID do Chamador), você precisará atribuí-lo ao seu assistente de IA. O processo é o mesmo se você estiver usando o número para chamadas de entrada ou saída.

## Atribuindo um Número

1. Vá para **Assistentes** → **(Seu Assistente)** → **Configurações**
2. Em **Números de Telefone**, selecione um número do seu inventário Autocalls.ai
3. Clique em **Salvar** para aplicar as alterações

## Uso do Número

Uma vez atribuído, o número pode ser usado para:

- **Entrada**: Chamadores discando este número alcançarão seu assistente de IA
- **Saída**: O número aparecerá como ID do Chamador para campanhas de saída
- **Ambos**: O mesmo número pode lidar com chamadas de entrada e saída

## Verificação de Número

Para números não alugados do Autocalls.ai:

- Você precisará verificar a propriedade via **código OTP** ou uma **chamada de verificação**
- Uma vez verificado, o número pode ser usado para entrada e saída
- O status de verificação persiste entre diferentes assistentes

## Testando

Antes de entrar no ar:

1. Faça uma chamada de teste de entrada para verificar se o assistente atende
2. Execute uma chamada de teste de saída para verificar a exibição do ID do Chamador
3. Verifique quaisquer recursos específicos (transferências de chamadas, correio de voz, etc.)

---

**Dica:** Sempre teste as funcionalidades de entrada e saída após fazer qualquer alteração nas atribuições de números.
