# Fundamentos do Autocalls

## Tipos de Chamadas

1. Chamadas de Entrada: Chamadas recebidas pelo sistema
2. Chamadas de Saída: Chamadas iniciadas pelo sistema

## Campanhas

Campanhas são conjuntos organizados de chamadas com objetivos específicos e públicos-alvo.

## Leads

Leads são clientes potenciais ou contatos que são direcionados em uma campanha.

## Configuração do Assistente

- Motor: Modelo de IA que alimenta a conversa
- Prompt: Instruções iniciais para o assistente de IA
- Mensagem Inicial: Primeira mensagem falada pelo assistente
- Idioma: Idioma usado para a conversa
- Voz: Voz selecionada para o assistente
- Ferramentas: Funcionalidades adicionais disponíveis para o assistente
- Extrações Pós-chamada: Dados extraídos após a chamada para análise

## Plataforma de Automação

Uma plataforma no-code para configurar e gerenciar chamadas automáticas.

## Provisionamento de Números de Telefone

Processo de aquisição e configuração de números de telefone para uso em campanhas.

## Custos

- Custo do Clinicalls por minuto: Taxa pelo uso do serviço de chamadas automáticas
- Custo da Twilio: Taxas adicionais pelo uso dos serviços de comunicação da Twilio

## Ferramentas

- Transferir Chamada: Capacidade de transferir a chamada para um agente humano
- Encerrar Chamada: Opção para terminar a chamada
- Integração de Calendário Cal.com: Funcionalidade de agendamento integrada com Cal.com
