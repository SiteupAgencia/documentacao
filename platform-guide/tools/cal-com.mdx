---

## Configurar Cal.com

1. Vá para o site [Cal.com](https://cal.com/) e crie uma conta
2. Crie um novo tipo de evento, por exemplo "Reunião com assistente de IA"
3. Copie o ID do evento clicando no tipo de evento e copiando este número da URL


4. Crie uma chave de API no Cal.com

## Adicionar a ferramenta ao seu assistente

Adicione esta ferramenta ao seu assistente e passe o ID do evento e a chave de API.
